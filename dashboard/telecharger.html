1️⃣

Télécharger l'APK

Cliquez sur le bouton ci-dessus pour télécharger le fichier APK sur votre téléphone Android.

2️⃣

Autoriser l'installation

Android demandera d'autoriser les sources inconnues. Allez dans Paramètres → Sécurité et activez "Sources inconnues" ou "Installer des apps inconnues".

3️⃣

Saisir le code entreprise

Au premier lancement, choisissez "Entreprise" et saisissez le Code Entreprise fourni par votre administrateur.
