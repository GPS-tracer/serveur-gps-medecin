📖 Guide Administrateur Suprême

1 Supervision du serveur

Le premier bloc affiche l'état du serveur Render en temps réel (ping toutes les 15 secondes). Si le serveur est rouge, il est en cours de réveil (cold-start ~30 secondes sur le plan gratuit). Le bloc Firebase confirme la connexion à la base de données.

2 Vue Globale — Tous les clients

Ce tableau liste tous les comptes inscrits sur la plateforme. Vous pouvez filtrer par nom, email, rôle ou abonnement. Le bouton 💣 envoie une commande de destruction à distance vers un agent spécifique. Les boutons ✓ Valider et ✕ Révoquer modifient le statut de partenariat d'une société.

3 Destruction à distance (Antivol Avancé)

Quand un agent force la désinstallation, une alerte apparaît automatiquement ici. Vous pouvez déclencher la destruction de trois façons :

  • ▸ Depuis une alerte : cliquer sur "💣 Détruire" sur la carte d'alerte
  • ▸ Depuis le tableau clients : cliquer sur 💣 sur la ligne de l'agent
  • ▸ Manuellement : saisir l'UID ou Device ID + raison + cliquer "Activer la Destruction"

⚠️ Une double confirmation est obligatoire : vous devrez taper DÉTRUIRE pour valider. Cette fonctionnalité est payante (option "Antivol Avancé").

4 Gestion des licences Chariow

Importer : collez les clés reçues de Chariow (une par ligne au format XXXX-XXXX-XXXX-XXXX), sélectionnez le type de pack et cliquez "Importer".

Générer : créez des clés de test pour valider le système. Ces clés sont fonctionnelles mais identifiées comme "test".

Le tableau Historique affiche toutes les clés avec leur statut (disponible/utilisé) et le client qui les a activées.

5 Partenariats & Validation de comptes

Recherchez une société par email, UID ou nom. Le bouton Valider attribue le rôle "partner" — permet des avantages tarifaires. Le bouton Révoquer retire ce statut. Utile pour les écoles communautaires, institutions publiques et partenaires commerciaux.

6 Cron & Notifications d'expiration

Cron d'expiration : vérifie tous les abonnements expirés et les désactive. S'exécute automatiquement à 01h00 UTC chaque nuit, mais vous pouvez le lancer manuellement ici.

Notifications d'expiration : envoie des rappels email aux sociétés dont l'abonnement expire dans 7, 3 ou 1 jour.
